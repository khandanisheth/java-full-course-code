Each folder focuses on a specific concept:

| Folder Name         | Description                                 |
|---------------------|---------------------------------------------|
| `Array`             | Programs based on arrays                    |

| `LambdaExpression`  | Lambda expressions                          |
| `OOPs`              | Object-Oriented Programming                 |

| `Ruf`               | Rough/test code                             |

| `String`            | String handling in Java                     |
